## 使用方法

### 安装

1. 在无名杀创建扩展，名为“忽悠宇宙” 。
2. 在无名杀 `extension` 目录下找到 `忽悠宇宙` 文件夹打开，其中有 `extension.js` 文件。
3. 下载《忽悠宇宙》扩展包，打开压缩包，打开有 `extension.js` 的文件夹界面。
4. 将上一步中的文件全部复制并粘贴至无名杀目录 `忽悠宇宙` 文件夹内，全部覆盖替换。
5. 重启游戏。
